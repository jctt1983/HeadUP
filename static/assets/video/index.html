Controls

										
Enable or disable playback controls, by default it's enabled.

										


										
Autoplay

										
Enable or disable autoplay when video is launched, by default it's enabled.

										

										
Note: autoplay may not work on some smartphone devices due to restrictions put in place on the browser of choice.


										
Resume

										
Automatically resume the video from the last time it was stoped or closed, by default it's enabled.

										


										
Analytics

										
Track video progress and send the progress to google analitics, by default it's enabled.

										

										

											Note: Google Analytics tag code should be present in the page for this feature to work.
										


										
Analytics Id

										
Set an id to identify the stats generated by the video in google analytics reports, by default it's
											empty.

										

										
If no id is specificied, the script will generate an id based on the name of the video file.


										
Quality

										
It allows you to select the default quality level when the video is opened, by default it's selected
											as auto.

										
This feature will only work for HLS videos or videos with format m3u8.

										

										

											options:
											
    
												
  • high: select the highest quality available
    • 
												
  • auto: automatically select a quality level between the highest and the
													lowest
    • 
												
  • low: select the lowest quality available
    • 
											

										


										
Hot keys

										
Enable or disable the hot keys when the video player has the focus, by default it's enabled.

										

										

											
    
												
  • Space bar toggles play/pause.
    • 
												
  • Right and Left Arrow keys seek the video forwards and back.
    • 
												
  • Up and Down Arrow keys increase and decrease the volume.
    • 
												
  • M key toggles mute/unmute.
    • 
												
  • F key toggles fullscreen off and on. (Does not work in Internet Explorer, it seems to be a
													limitation where
													scripts
													cannot request fullscreen without a mouse click)
    • 
												
  • Double-clicking with the mouse toggles fullscreen off and on.
    • 
												
  • Number keys from 0-9 skip to a percentage of the video. 0 is 0% and 9 is 90%.
    •
